1. First run the server with ./server.sh
2. Then open two different terminals. From each terminal execute ./client.sh

> The server will start playing music after both of clients have connected.
> The music will be stopped and the music.txt file will be updated when both of clients disconnect.

Controls:
next: go to next song
previous: go to previous song
exit: disconnect from server


Video demo: https://www.youtube.com/watch?v=hpbEcW-NwCk
